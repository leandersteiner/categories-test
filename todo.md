# Categories in ecommerce project

## Features

1. Users can have multiple shops
2. Users can have multiple products
3. Products are global for all shops
4. Users can have multiple categories
5. Products can have multiple categories
6. Categories can be nested
7. Users can have multiple Collections
8. Products can be in multiple collections
9. Collections can be nested
10. Products can be in multiple categories
11. Products can be in multiple shops
12. Products can be in multiple collections
13. Each shop can have multiple collections
14. If a shop has no collections, it will show all products
15. Collections are the top level navigation
16. Categories are the second level navigation

This will be a minimal implementation to just show the basic functionality of the ecommerce system.
We don't need to have authentication and authorization in this project or users.
The starting point expectation is that we are already logged in.

## Backend 

1. Language: Golang with net/http and in memory repositories

## Frontend

1. React with Typescript
2. Any fitting UI framework
3. React Query for data fetching
